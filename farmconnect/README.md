# FarmConnect - Rural Kenya E-Commerce Platform

A complete responsive e-commerce website for selling farm products in rural Kenya, supporting multiple farmers listing their products with user registration, product upload, cart functionality, and mobile-friendly design.

## Features

### User Features
- User registration and authentication
- Browse products by category
- Search functionality
- Shopping cart
- Checkout system
- Mobile-responsive design

### Farmer Features
- Farmer dashboard
- Product management
- Order management
- Sales analytics

### Admin Features
- User management
- Product approval system
- Platform analytics
- Content moderation

## Tech Stack

### Frontend
- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Responsive design

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT authentication
- File upload with Multer
